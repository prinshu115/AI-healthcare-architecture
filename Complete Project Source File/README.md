
# Agentic Healthcare AI Platform (CrewAI)

This repository demonstrates a production-oriented agentic AI system
designed for healthcare use cases using CrewAI.

The focus is on orchestration, safety, and human oversight rather than
raw model capability.

## Use Case
- Medical research summarization
- Clinical insight generation
- Safety and compliance review

## How to Run
pip install -r requirements.txt
python main.py
